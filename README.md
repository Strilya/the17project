## üîÑ How The Rotation System Works

### Smart History Tracking

1. **Style History**
   - Tracks last 2 styles used
   - Never picks a style you just used
   - Example: If you used Style 1, next video picks from Styles 2-5

2. **Background History**
   - Tracks last 5 backgrounds used
   - Avoids same video footage recently used
   - Even within same category, picks different clips
